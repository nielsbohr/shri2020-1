# Первое тестовое задание ШРИ 2020

## Stack

* SASS(scss)
* webpack + webpack-dev-server
* eslint, stylelint

## Точки входа

### Development

`npm run dev`

Entry: dev.js \
Webpack config: webpack.dev.js

### Production

`npm run build`

Entry: prod.js \
Webpack config: webpack.prod.js

### Server

`npm start`

app.js - Express based приложение для проверки билда.
